MUY IMPORTANTE
PARA CLAVE UNICA SE DEBE HACER LO SIGUIENTE....
1. INSERTAR EL BOTON DE CLAVE UNICA, EN ESE BOTON HAY QUE REDIRECCIONAR O ENVIAR LA INFO A LA API
2. DESDE LA API SE CONSTRUYE EL PRIMER GET CON LOS PARAMETROS DE CLIENT_ID, ETC, SE HACE LA PETICIÓN.
3. CUANDO LA API RESPONDA, HAY QUE MODIFICAR LA RESPUESTA PARA AGREGAR LAS URL EN EL FORM() DE FORMA CORRECTA
   YA QUE LE FALTA LA PRIMERA PARTE DE LA URL DE CLAVE UNICA (VER INDEX.HTML DE LA CARPETA CLAVE_UNICA DEL
   ESCRITORIO).
4. EJECUTAR EL SEGUNDO POST PARA OBTENER EL TOKEN(ACÁ YA ESTÁ VALIDADO EL USUARIO)
5. EJECUTAR EL TERCER POST PARA OBTENER DATOS INICIALES (NOMBRE COMPLETO, RUN)
